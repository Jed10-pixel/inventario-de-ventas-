# 📦 Sistema de Inventario de Ventas

Este es un proyecto desarrollado en **Python y HTML** diseñado para gestionar el inventario y controlar las ventas de un negocio de manera sencilla y eficiente.

## 🚀 Características
* **Control de Stock:** Permite visualizar los productos disponibles en el inventario.
* **Registro de Ventas:** Interfaz interactiva para procesar las ventas diarias.
* **Interfaz Web:** Diseñada con páginas HTML (`inicio.html`, `índice.html`) para una navegación cómoda.

## 🛠️ Tecnologías utilizadas
* **Backend:** Python 3.13
* **Frontend:** HTML5 / CSS3

## 💻 Cómo ejecutar el proyecto

1. Descarga o clona este repositorio.
2. Asegúrate de tener Python instalado.
3. Ejecuta el archivo principal desde tu terminal:
```bash
   python aplicación.py
